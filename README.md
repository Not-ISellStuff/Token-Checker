# Token Checker
The token checker has a very op proxy scraper 🥶 (I definitely did not use proxy scrape's api)
This is my first tool in ruby ive only been coding in ruby for 2 days so it's not very good

# How To Setup
1. Install Ruby From https://rubyinstaller.org/downloads/
![image](https://github.com/Not-ISellStuff/Token-Checker/assets/174392963/d9feccc8-feaa-4d49-9b24-2b438f48349a)
2. Run setup.bat it will install httpx which is the lib I used to make the token checker
3. put your tokens in tokens.txt
and thats it just open main.rb and you can start checking tokens
